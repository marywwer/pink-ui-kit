# 🌸 Fleur

**Fleur** — учебный интернет-магазин цветочных букетов, разработанный на **React + Vite + TypeScript** без использования UI-библиотек. Проект демонстрирует построение современного SPA-приложения с авторизацией, ролевой моделью, защищёнными маршрутами и работой с локальным хранилищем.

## Возможности

### Для всех пользователей

* Просмотр каталога букетов
* Поиск и фильтрация товаров
* Просмотр карточки товара
* Просмотр отзывов покупателей
* Адаптивный интерфейс

### Для авторизованных пользователей

* Регистрация и вход в аккаунт
* Авторизация по токену
* Добавление товаров в корзину
* Добавление товаров в избранное
* Оставление отзывов
* Автоматическое восстановление сессии после перезагрузки страницы

### Для администратора

* Доступ к панели администратора
* Изменение статуса товара ("В наличии" / "Нет в наличии")
* Просмотр статистики товаров
